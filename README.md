VIN Decoder SPA
Посилання на розгорнутий застосунок

Live Demo: https://abp-test-task.vercel.app/

Інструкція з локального запуску

Клонувати репозиторій:

git clone https://github.com/KirillVGX/ABP-test-task.git

Встановити залежності:

npm install

Запустити застосунок:

npm run dev

Після запуску застосунок буде доступний за адресою:
http://localhost:5173
